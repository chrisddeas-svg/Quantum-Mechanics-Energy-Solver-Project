Quantum Mechanic Energy Solver


The project is made up of three parts displayed in each of the branches;

"Analytical-Evaluation" 
A branch which contains a document that tests the accuracy of the computational solver given a set of specific variables. 

"Coded-Solver" 
A branch which contains the python 3 code that makes up the energy solver. 

"Derivations" 
A branch which has three documents each displaying a seperate derivation for the specific potencial function. With the enrgy equation then being utilised in the analytical evaluation.
